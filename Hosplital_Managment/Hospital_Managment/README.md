#Project Description: Hospital Bed Management System



This is a console-based Hospital Bed Management System built in C.

The program offers a straightforward way to manage bed occupancy and keep track of basic patient information within a hospital ward of a fixed size (100 beds).

🛏️ Key Features
Patient Management: Quickly admit new patients to an available bed and discharge patients when they leave.

Real-time Bed Status: View the ward layout with a visual 2D grid where:

'X' represents an Occupied bed.

'O' represents an Empty bed.

Statistics and Reporting: Access immediate data on the ward's performance, including:

Total bed count and the number of available beds.

Current bed occupancy percentage.

Details of the last patient admitted and the last patient discharged.

Robust Input: Features basic input validation with multiple attempts to ensure more reliable user interaction.
