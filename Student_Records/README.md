# Student_Records


This project is a simple **Student Record Management System** implemented in **C language** using **Linked Lists**.  
It allows users to dynamically manage student data such as name, roll number, course, and marks.  
The system provides full CRUD (Create, Read, Update, Delete) operations on a linked list.

---

## 🧠 Features

✅ Add a student at:
- Beginning  
- End  
- Specific position  

✅ Delete operations:
- From beginning  
- From end  
- From a specific position  
- Delete all students  

✅ Display:
- Show all student records  
- Count total number of students  

✅ User Interaction:
- Interactive menu-driven interface  
- Continuous loop until user exits
